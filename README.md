# vim-lunarized

A color scheme based on Solarized with neutral base colors.


## Configuration

```vim
set background = dark
colorschme lunarized
```

## Options

Export variables and functions so custom highlights can be added

```vim
let lunarized_export = 1
```

Force color mode

```vim
let lunarized_color_mode = "rgb" / "256" / "16"
```
